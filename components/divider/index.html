Divider (分割线) ​

概述 ​

这是 Divider (分割线)组件的文档，用于在内容中进行视觉分割，支持水平、垂直分割线及多种样式。

预览 ​

基本使用
最基础的 Divider 用法
<template>
  <div>
    <!-- 基本水平分隔线 -->
    <h3>基本水平分隔线</h3>
    <se-divider />

    <!-- 虚线分隔线 -->
    <h3>虚线分隔线</h3>
    <se-divider dashed />

    <!-- 垂直分隔线 -->
    <h3>垂直分隔线</h3>
    <div style="display: flex; align-items: center;">
      Text <se-divider type="vertical" /> Text
    </div>

    <!-- 带内容的分隔线 -->
    <h3>带内容的分隔线</h3>
    <se-divider content="Left Content" orientation="left" />
    <se-divider content="Center Content" />
    <se-divider content="Right Content" orientation="right" />

    <!-- 插槽内容的分隔线 -->
    <h3>插槽内容的分隔线</h3>
    <se-divider orientation="center">
      <strong>Custom Content</strong>
    </se-divider>
  </div>
</template>

基础配置 ​

Divider 组件提供了丰富的配置选项，支持多种样式、对齐方式和自定义内容。

Divider参数 ​

参数名类型默认值描述跳转 Demo
typeStringhorizontal分割线方向，可选值为 horizontal 或 vertical基本使用
dashedBooleanfalse是否为虚线样式虚线分割线
contentString""分割线上的内容（优先级低于插槽）带内容的分割线
orientationStringcenter分割线内容的对齐方式，可选值为 left、center 和 right带内容的分割线
classString / Array / Object""自定义分割线的类名自定义样式
styleObject{}自定义分割线的内联样式自定义样式

其他说明 ​

  • 插槽优先级
    如果同时传入 content 和使用默认插槽，插槽内容会优先显示。

  • 扩展性
    支持通过 class 和 style 定制更多样式，同时保持原有功能。

  • 兼容性
    确保组件在现代浏览器中表现一致，低版本浏览器可能需要添加 polyfill。
