# Mini Message(微型消息)

这是 `MiniMsg` (微型消息)组件的文档。
## 基础用法
**从 `selab-ui` 引入**
`MiniMsg`注册了一个全局方法 `$seMiniMsg`，也可以单独引入。

```js
import { SeMiniMsg } from 'selab-ui';
```

`MiniMsg` 方法接受一个 `option对象` 作为其参数，通过设置其 `message` 属性设置文本内容。
