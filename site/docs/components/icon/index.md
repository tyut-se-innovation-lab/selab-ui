### Icon 参数
| 参数名        | 类型                       | 默认值       | 描述                                       | 
|:-----------| :------------------------- |:----------| :----------------------------------------- |
| `name`     | String                     | down      | 图标名称，用于生成图标类名。               | 
| `color`    | String                     | '#333'    | 图标颜色。                                 | 
| `iconSize` | Number, String             | 'inherit' | 图标大小。支持数字或字符串形式。           |
